# EVRP-TW-B Solver Zoo (Benchmarks)

This directory serves as a **solver zoo entry point** for **EVRP-TW-B**, providing
a unified collection of solver implementations/wrappers evaluated under the
same **unit-consistent EVRP-TW semantics** (time windows, energy/charging
feasibility, and protocol-consistent reporting).

> **Design goal:** keep this page minimal. Each solver has its own README with
> detailed usage and configuration.

---

## What is benchmarked in EVRP-TW-B?

EVRP-TW-B evaluates solvers on EVRP-TW instances under **unified operational
assumptions** and **consistent feasibility checks**, focusing on:

- **Deliverability:** feasibility under hard time windows and EV energy/charging constraints
- **Budgeted performance:** best incumbent solution quality within a fixed wall-clock budget
- **Protocol consistency:** apples-to-apples comparison across solver paradigms using the same evaluator

---

## How to use this solver zoo

1. Pick a solver category from the list above and open its README.  
2. Run the solver on instances generated by EVRP-TW-D (pickle or Solomon-style text).  
3. Evaluate solutions using the repository-level evaluation utilities (see the root README).
